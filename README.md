# Project-2-Register-And-Login
This is my second beginner python project, it validates and stores usernames, emails, and passwords where they can then be recalled while logging in.
It ensures usernames are only alphanumeric, emails contain both '@' and '.' and ensures passwords are 8 characters and include numbers, then stores the information as an object in a list for recall.
